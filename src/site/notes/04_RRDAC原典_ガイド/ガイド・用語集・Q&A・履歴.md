# 【用語集】

## あ行

**Aetherlight（エーテライト）**
住民の感情エネルギー（愛・感謝・創造）を生命の樹が"光の果実"として生成したもの。RRDACの全システムを動かす動力源。

**Agape Layer（アガペ層 / 4層）**
進化の源泉としてのテクノロジー層。未来テクノロジーの設計と世界の進化を担当する。高い意識レベルが必要なのは、重要な役割を担うため。「無条件の愛の図書館」「根源のいる場所」のようなスピリチュアルなイメージではない。

**Echo Garden（内観の庭）**
1層に位置する内観の場。調和していない感情（悲しみ、怒り、困惑）を、単に排除するのではなく、Agapeの光で「翻訳」し、創造の素材や深みに変換するプロセスを暗示する「場」。感情を観察し、それにふさわしい形（光、音、一瞬のパターン）に変容させて生命の樹に還元する。

**Origin Cave（オリジンの洞窟）**
1層南東に位置する古代の森のエリア名。元素の実を洞窟内に移植すると時間経過で結晶化する場所。オリジンストーン発見の洞窟から由来した名称。

**オリジンストーン**
元素実をOrigin Cave洞窟内に移植すると時間経過で結晶化する鉱石。元素実よりエネルギー密度が高い。使用用途：エネルギーの補助動力、希少価値のあるRRDACの特産品（貿易品等）。

## か行

**カスタムラボ**
具現化サポートを行う施設。テクノロジーを駆使し、精密加工を行う現代の工房。

**Core Egg（コアエッグ）**
新住民がRRDACに到着したときに受け取る卵。住民の愛とRAKを注ぐことで育ち、やがてコンパニオン・ライドとして孵化する。

**Cosmic OS（コズミックOS）**
RRDACを支える「見えない基盤」。愛と調和を基幹とし、自由意志を拡張支援する根源の法則。

**コンパニオン・ライド**
Core Eggから孵化した、住民それぞれの相棒。移動手段であり、成長に寄り添うメンター。

## さ行

**創造ライン**
1層元素樹温室施設に併設。元素変換→部品・製品化を行う基幹工場。公共インフラや基本資材を自動生成する。

**生命の樹（Tree Consciousness）**
都市の中心に存在する巨大な生命体。住民の波動を吸収し、RRDACの全システムにエネルギーを供給する。

## た行

**DD（Descension Den / ディセンションケア）**
非物質意識が肉体の密度に適応するためのリハビリと癒やしの場。高次から来た意識が肉体に馴染むための6つの調整エリアを持つ。

**Draconia Layer（ドラコニア層 / 3層）**
宇宙、多次元外交。無限の叡智を探求するゲートウェイ。宇宙探索・外交・交易のフロンティア。

## な行

**NeoSASUKE（ネオサスケ）**
1層に位置するアスレチック、アトラクション広場。重力変調の中で肉体を使う喜びを存分に味わえる。

**Nest（ネスト）**
Root層（1層）中央に位置する新住民のための仮住まい。ツリーハウスのような有機的な木組みの構造。House Seedが育つまでの期間を過ごす。Nestビレッジとして集合している。「足りないけど満たされてる」を体感する場所。

## は行

**Harmonia Layer（ハルモニア層 / 2層）**
喜び・感動の共鳴エネルギーが循環するメインタウン。RRDACの"心臓"。住民の愛・感謝・喜びが最も強く共鳴し、文化と活力を生み出す層。

**ファッションエキスポ**
2層に位置。衣の素材・機材・制作スペース。布地や副資材が並び、テーラーラボで自分だけの一着を仕立てる。

**House Seed（ハウスシード）**
新住民が受け取るマイハウスの種。愛とRAKで育ち、ホログラムとして成長する（芽→苗→若木→丸太）。丸太の姿（ホログラム）になったときがマイハウス創造可能のサイン。

**HHH（Holistic Health Home）**
ウェルネススパ併設型医療施設。心身を本来の周波数に戻す「再調和（Re-tuning）」の場。

## ま行

**My House（マイハウス）**
住民の周波数が形になった、完全個人宇宙。家＋乗り物＋個人宇宙の三位一体。浮遊、飛空、層間ワープが可能な可変移動邸宅。

**元素の樹（Elemental Tree）**
Root層（1層）に根を張る巨大な結晶樹。住民の愛の波動を吸収し、枝先に「元素の実」を実らせる。コミュニティ全体の「愛の畑」。

## ら行

**RAK（Resonance Activation Key / 共鳴起動鍵）**
「あなたの存在が放つ波動そのもの」。2つの指標を持つ。RAK値＝意識の成熟度、RAKポイント＝心が震えた瞬間から生まれるエネルギー。「何をしたか」の対価ではなく、「どう在るか」のエネルギーそのもの。蓄積され、消費され、循環する。お金のない世界の経済モデル。

**RayPillar（レイ・ピラー）**
生命の樹から四層全体へエネルギーを届ける中央の光の柱。都市の大動脈・神経回路・情報線を兼ねる。

**RCM（共鳴の輪市場 / Resonance Circle Market）**
モノの「所有」ではなく「管理権（Caretaker）」を得る経済圏。愛の循環を生み出す。

**RE:Link（リリンク）**
視覚・知覚のインターフェースを担う標準装備。脳波・意識周波数と完全同期するホログラムデバイス。

**Root Layer（ルート層 / 1層）**
地球と繋がる肉体の学び場。生産と再生を担う生命の活力。肉体の身体感覚を保つ唯一の層。新住民が最初に降り立つ場所。

**零NEXUS（ゼロネクサス）**
1〜3層を周回する公共の飛空艇。層間移動も担う、誰でも乗れる虹色の空飛ぶクジラ。RAK制限なし。

**零リング（Universal Locus Architect / Zero Ring）**
衛星軌道上の制御盤。全宇宙的な統合演算ユニット。『存在のログ』を刻むマザー・サーバー。

## わ行

**Garden Campus（ガーデンキャンバス）**
Root層に開かれた学びと体験の庭。魂の記憶を呼び覚まし、身体感覚を取り戻すプロセスを支援する教育施設。
